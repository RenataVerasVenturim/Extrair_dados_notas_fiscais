# Extrair_dados_notas_fiscais
Extrair dados de notas fiscais com Tesseract OCR e expressões regulares


REGEX utilizado para extrair número da nota fiscal = N[°º]?\s?\d{1,10}(?:\.\d{3})*
Site utilizado: https://regex101.com/
